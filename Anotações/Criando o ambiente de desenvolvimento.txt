 Conhecendo a CLI do .NET
	- Conhecendo a CLI do .NET
		- CLI (Command-Line Interface)
			- Processa os comandos que serão enviados para um programa de 
			computador na forma de linhas de texto.
			- Comandos .NET
				- dotnet --version (Verificar versão)
				- dotnet --help (apresenta alguns comandos e descrições)
				- dotnet --info (informações sobre o dotnet, SO, local de instalação)
				- dotnet new (criação de projetos)
				- dotnet new --list --tag web (lista tipos por tag)
				-- dotnet new --help (apresenta alguns comandos relacionados ao new)
				-- dotnet new console -o myApp (Cria novo projeto com tipo console)
				-- dotnet run (Executa o projeto)
				-- dotnet build (Compila o projeto)
				-- dotnet restore (Restore de projetos com as dependencias perdidas/apagadas)
