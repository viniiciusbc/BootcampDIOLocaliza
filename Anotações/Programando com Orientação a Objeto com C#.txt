 Conhecendo a Programação orientada a Objetos
	- Apresentando a programação orientada a objetos
		- A POO é um paradigma de programação, ou seja, corresponde
		a uma técnica de programação para um fim específico.
		- Dentro desta técnica, existem quatro pilares:
			- Abstração 
			- Encapsulamento
			- Herança
			- Polimorfismo
		- O principal conceito da POO são classes e objetos 
			- Classes
				- Molde com elementos que fazem sentido para o contexto
			- Método 
				- Ação quem o objeto fará
			- Objetos
				- Objeto é a construção de uma classe, uma classe materializada 
		
	- Tipos de Paradigmas 
		- Paradigma de programação é diferente de linguagem de programação
		- Uma linguagem de programação implenta um ou mais paradigmas 
		- Um para nada mais é do que um modelo de técnicas, estruturas e formas
		de solucionar um problema
		- Tipos de paradigmas
			- Programação orientada a objetos (estudando atualmente)
			- Programação estruturada
			- Programando imperativa
			- Programação procedural 
			- Programação orientada a eventos
			- Programação lógica
